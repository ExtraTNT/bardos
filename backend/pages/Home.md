# Welcome to Bardos

A tiny **markdown-driven** website rendered with [odocosJS](https://github.com/ExtraTNT/odocosJs).

---

## Features

- Headings, paragraphs, and inline formatting
- **Bold**, *italic*, and `inline code`
- [Links](https://example.com) and images
- Ordered & unordered lists
- Blockquotes and fenced code blocks
- Spoiler tags
- Syntax highlighted code blocks

> This whole page is written in plain markdown and served from the filesystem.

## Quick Example

```js
const greeting = 'Hello, world!';
const add = a => b => a + b;
```

1. Write markdown files
2. Drop them in the pages directory
3. The backend serves them, the frontend renders them
4. Done!
