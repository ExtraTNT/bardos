# Code Highlights

Browse syntax-highlighted code samples. The highlighter is **plugin-based** — each language is a list of `[regex, className]` pairs fed to a generic tokenizer.

---

## Available Languages

- **JavaScript** — full es6+ coverage
- **Haskell** — keywords, types, comments
- **C** - keywords, types, comments
- **Csharp** - keywords, types, comments
- **Java** - keywords, types, comments
- **Rust** - keywords, types, comments
- **Python** - keywords, types, comments
- *Any other* — rendered as plain text (graceful fallback)

Use the sub-nav to pick a language.
