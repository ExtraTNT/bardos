# Image Demo

This page demonstrates loading images from the API endpoint.

---

## Coffee Example

![Coffee cup](/api/images/coffee.png)

The image above is served by:

- `GET /api/images/coffee.png`

You can also fetch the image index with:

- `GET /api/images`

## Notes

- Image markdown uses the standard syntax: `![alt text](url)`
- The backend now supports image MIME types and path validation
